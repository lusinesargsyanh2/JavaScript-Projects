# Array Methods Implementation

A collection of custom JavaScript implementations of common `Array.prototype` methods.

## Implemented Methods

- `myForEach`
- `myMap`
- `myFilter`
- `myReduce`
- `myFind`
- `myIncludes`
- `mySlice`
- `mySplice`
- `myPush`
- `myPop`
- `myShift`
- `myUnshift`
- `myFlat`
- `myEvery`
- `myFindIndex`
- `mySome`
